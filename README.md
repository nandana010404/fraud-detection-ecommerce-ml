# Fraud Detection in E-Commerce Transactions

## Overview
This project is a machine learning-based web application developed to detect fraudulent e-commerce transactions. The system predicts whether a transaction is fraudulent or legitimate using trained machine learning models.

## Features
- Fraud detection using Machine Learning
- Stacking Classifier prediction
- XGBoost model training
- User-friendly Flask web interface
- PDF report generation
- Transaction analysis

## Technologies Used
- Python
- Flask
- Scikit-learn
- XGBoost
- HTML/CSS/JavaScript

## Machine Learning Models
- XGBoost Classifier
- Stacking Classifier

## How to Run

1. Clone the repository

2. Install dependencies
pip install -r requirements.txt

3. Run the Flask application
python app.py
